## 🐛 Troubleshooting

### Problem: npm install fails
**Solution:**
```bash
npm cache clean --force
npm install
```

### Problem: Port 3000 in use
**Solution:**
- App will auto-use port 3001
- Or set manually: `set PORT=3002 && npm start`

### Problem: Animations laggy
**Solution:**
- Refresh browser (Ctrl+F5)
- Close other tabs
- First load is always slower

### Problem: Page blank
**Solution:**
- Check browser console (F12)
- Look for errors
- Ensure all files present
- Try: `npm install` again

### Problem: Can't find npm
**Solution:**
- Install Node.js from nodejs.org
- Restart terminal
- Try again
