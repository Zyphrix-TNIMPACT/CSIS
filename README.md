## ЁЯЪА Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Setup Steps

1. **Install Dependencies**
```bash
npm install
```

2. **Start Development Server**
```bash
npm start
```

3. **Build for Production**
```bash
npm build
```

The application will open at `http://localhost:3000`
