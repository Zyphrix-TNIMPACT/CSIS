# 🚀 CSIS Quick Start Guide

## Installation (3 Simple Steps)

### Step 1: Install Dependencies
Open terminal in the CSIS folder and run:
```bash
npm install
```

### Step 2: Start the Application
```bash
npm start
```

### Step 3: View in Browser
The app will automatically open at: `http://localhost:3000`

---

## 🎬 Demo Flow

### 1. Splash Screen (4 seconds)
- Watch the CSIS logo animation
- Camera scan effect
- Progress bar loading
- Auto-transitions to login

### 2. Login Page
- Enter any username/password
- Click "LOGIN" button
- Or use "EMERGENCY ACCESS" for supervisor mode

### 3. Dashboard (Main Screen)
- View 4 live camera feeds
- See real-time detection boxes
- Watch risk scores animate
- Check live alerts panel
- Review incident log

### 4. Navigation
Use the top navigation bar:
- **Dashboard** - Main control center
- **Cameras** - Individual camera views
- **Incidents** - Incident history
- **Analytics** - Risk analytics
- **Settings** - System configuration

## 🔧 Troubleshooting

### If npm install fails:
```bash
npm cache clean --force
npm install
```

### If port 3000 is busy:
The app will automatically use port 3001 or ask you to use another port.

### If animations are slow:
This is normal on first load. Refresh the page for smoother performance.

## 💡 Customization Tips

### Modify Alerts
Edit alert data in Dashboard component

### Adjust Animations
Modify Framer Motion parameters in component files
